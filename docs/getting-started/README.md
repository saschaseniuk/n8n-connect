## Overview

n8n-connect consists of two main packages:

1. **@n8n-connect/core** - Framework-agnostic core functionality
2. **@n8n-connect/react** - React/Next.js integration (depends on core)

Most users will want to install `@n8n-connect/react`, which includes core as a dependency.

## Prerequisites

- Node.js 18+ or compatible runtime
- An n8n instance with webhook-triggered workflows
- TypeScript 5.0+ (recommended)
